# 환묘 케어 일기 모바일 웹 프로젝트 계획

## 📋 프로젝트 개요
- **목적**: 고양이 케어 일기를 작성하고 관리하는 모바일 웹 애플리케이션
- **특징**: 개인용 일기 (공유 기능 없음), 데이터 로컬 저장

## 🎯 핵심 기능

### 1. 일기 작성 기능
- 날짜별 일기 작성
- 오늘 날짜 자동 선택
- 다양한 입력 필드:
  - 기본 정보: 체중, 건강 상태, 식사량
  - 활동 기록: 산책, 놀이 시간
  - 특이사항: 메모, 사진 첨부
  - 기분/상태: 고양이 기분, 활동량

### 2. 일기 조회 기능
- 캘린더 뷰로 날짜별 일기 확인
- 리스트 뷰로 최근 일기 목록
- 상세 보기: 특정 날짜의 전체 기록

### 3. 데이터 저장
- 로컬 스토리지 또는 IndexedDB 사용
- 브라우저에 데이터 영구 저장
- 데이터 백업/복원 기능 (선택사항)

### 4. 통계 및 분석 (선택사항)
- 체중 변화 그래프
- 건강 상태 트렌드
- 월별 요약

## 🛠 기술 스택 제안

### 옵션 1: React 기반
- **프레임워크**: React + Vite
- **스타일링**: Tailwind CSS 또는 Styled Components
- **상태 관리**: Zustand 또는 Context API
- **데이터 저장**: IndexedDB (idb 라이브러리)
- **날짜 처리**: date-fns
- **아이콘**: React Icons

### 옵션 2: Vue.js 기반
- **프레임워크**: Vue 3 + Vite
- **스타일링**: Tailwind CSS
- **상태 관리**: Pinia
- **데이터 저장**: IndexedDB
- **날짜 처리**: date-fns

### 옵션 3: 순수 JavaScript (경량)
- **프레임워크**: Vanilla JS
- **스타일링**: CSS3 (모바일 우선)
- **데이터 저장**: IndexedDB (Native API)
- **빌드 도구**: Vite (최소 설정)

## 📱 UI/UX 설계

### 디자인 원칙
- **모바일 우선**: 터치 친화적 인터페이스
- **직관적**: 복잡한 기능 없이 간단한 조작
- **시각적**: 고양이 테마의 따뜻한 색상
- **접근성**: 큰 버튼, 명확한 레이블

### 주요 화면
1. **홈 화면**: 오늘 일기 작성 버튼, 최근 일기 목록
2. **일기 작성 화면**: 폼 입력 필드들
3. **캘린더 화면**: 날짜별 일기 표시
4. **상세 보기 화면**: 특정 날짜의 전체 기록

## 🚀 개발 단계

3. 레이아웃 컴포넌트 구성

2. 일기 저장/조회/수정/삭제 기능 구현

## 💡 추가 고려사항

### 보안 및 프라이버시
- 모든 데이터는 로컬에만 저장
- 외부 서버 전송 없음
- 브라우저 데이터 삭제 시 일기도 삭제됨 (백업 기능 고려)

### 사용자 경험
- 오프라인 작동 가능
- 빠른 로딩 속도
- 직관적인 네비게이션
- 데이터 입력 시 자동 저장

### 확장 가능성
