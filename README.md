# 🐱 환묘 케어 일기

고양이 케어 일기를 작성하고 관리하는 모바일 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📷 **사진 업로드**: 매일의 사진을 저장하고 관리
- 📝 **멘션 작성**: 오늘의 이야기를 자유롭게 기록
- ✅ **체크 항목**: 매일 체크해야 하는 항목들을 기록
  - 몸무게 측정
  - 먹은 사료량 (건사료/습사료)
  - 화장실 횟수 (감자/맛동산)
  - 구토 횟수 및 특이사항
  - 수액량
  - 투약 체크
- 💊 **약 관리**: 처방약과 보조제를 등록하고 관리
  - 약 이름, 용량, 메모 등록
  - 처방약/보조제 분류
  - 일기 작성 시 체크박스로 투약 기록
  - 등록/수정/삭제 기능
- 📅 **날짜별 관리**: 날짜별로 일기를 작성하고 조회
- 💾 **로컬 저장**: 모든 데이터는 브라우저에 안전하게 저장 (IndexedDB)
- 📱 **모바일 최적화**: 모바일 환경에 최적화된 UI/UX

## 🛠 기술 스택

- **React 18** - UI 라이브러리
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **React Router** - 라우팅
- **IndexedDB** - 로컬 데이터 저장
- **date-fns** - 날짜 처리

## 🚀 시작하기

### 로컬 개발

```bash

npm run dev
```

## 📱 PWA 기능

- 🏠 홈 화면에 추가 가능
- 📴 오프라인에서도 작동
- 🚀 앱처럼 사용 가능
- 🔄 자동 업데이트

## 📱 사용 방법

### 기본 사용
1. **홈 화면**: 오늘 일기 작성 버튼을 눌러 일기를 작성합니다.
2. **일기 작성**: 사진, 체크 항목, 투약 정보, 멘션을 입력한 후 저장합니다.
3. **일기 목록**: 전체 일기 목록에서 과거 일기를 확인할 수 있습니다.
4. **일기 상세**: 일기를 클릭하면 상세 내용을 확인하고 수정/삭제할 수 있습니다.

### 약 관리
1. **약 관리 메뉴**: 홈 화면에서 "💊 약 관리" 버튼 클릭
2. **약 추가**: "새 약 추가" 버튼으로 처방약/보조제 등록
3. **약 정보 입력**: 약 이름, 분류, 용량, 메모 입력
4. **투약 기록**: 일기 작성 시 체크박스로 투약 여부 기록

## 💡 특징

- 모든 데이터는 브라우저의 IndexedDB에 저장됩니다.
- 인터넷 연결 없이도 작동합니다.
- 다른 사람과 데이터를 공유하지 않습니다.
- 브라우저 데이터를 삭제하면 일기도 함께 삭제됩니다.

## 📝 데이터 구조

### 일기 데이터
- 날짜 (YYYY-MM-DD 형식)
- 사진 (Base64 인코딩)
- 멘션 (텍스트)
- 체크 항목
  - 몸무게 (kg)
  - 건사료량 (g)
  - 습사료량 (g)
  - 감자 횟수
  - 맛동산 횟수
  - 구토 횟수 및 타입
  - 수액량 (ml)
  - 투약한 약 ID 배열
- 생성/수정 시간

### 약 데이터
- ID (자동 생성)
- 약 이름
- 분류 (처방약/보조제)
- 용량
- 메모
- 생성/수정 시간

## 🔒 개인정보 보호

- 모든 데이터는 로컬에만 저장됩니다.
- 외부 서버로 데이터를 전송하지 않습니다.
- 브라우저의 개인정보 보호 설정을 따릅니다.
